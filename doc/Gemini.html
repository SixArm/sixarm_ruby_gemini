install_command(ops) click to toggle source

Create the system command to do a gem install

Options

  • name e.g. “foo” [required]
  • version e.g. “>=1.2.3“
  • trust: no|low|medium|high (aka NoSecurity|LowSecurity|MediumSecurity|HighSecurity)
  • append (aka installing): anything to append to the gem install command e.g. “ —no-ri —no-rdoc —no-user-install“

Return the command as a string 

     # File lib/sixarm_ruby_gemini.rb, line 131
131:   def self.install_command(ops)
132:     name,version,trust,source,append,installing = [:name,:version,:trust,:source,:append,:installing].map{|k| ops[k]||ops[k.to_s]}
133:     append||=installing
134:     return "sudo gem install #{name}" +
135:       (version ? " --version \"#{version}\"" : '' ) +
136:       (trust ? " --trust-policy #{TRUST_POLICY_NICKNAME[trust]||trust}" : '') +
137:       (source ? [*source].map{|x| " --source \"#{x}\""}.join : '') +
138:       (append ? (' '+append) : '')
139:   end
